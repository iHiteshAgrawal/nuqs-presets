# Next.js Dashboard Example

An admin dashboard with data tables and tab navigation demonstrating `nuqs-presets`.

## Features

- ✅ **Tab Navigation** - Type-safe tab switching
- ✅ **Data Tables** - Complete table with all features
- ✅ **Date Range Filtering** - Filter data by date ranges
- ✅ **Multi-select** - Select multiple items
- ✅ **Dashboard Layout** - Professional admin layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3002](http://localhost:3002) in your browser.
