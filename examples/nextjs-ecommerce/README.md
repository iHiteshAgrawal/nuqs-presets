# Next.js E-commerce Example

A comprehensive e-commerce product filtering interface demonstrating advanced usage of `nuqs-presets`.

## Features

- ✅ **Advanced Filtering** - Multi-faceted filtering (category, price range, brand, rating)
- ✅ **Debounced Search** - Fast search with debouncing
- ✅ **Multi-column Sorting** - Sort by multiple columns
- ✅ **Filter Badges** - Visual filter indicators with clear functionality
- ✅ **Responsive Design** - Mobile-friendly with Tailwind CSS
- ✅ **Type-safe** - Zod schema validation for filters

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.
