# React + Vite Example

A client-side React SPA demonstrating framework-agnostic usage of `nuqs-presets`.

## Features

- ✅ **Framework Agnostic** - Works outside of Next.js
- ✅ **Client-side Routing** - React Router integration
- ✅ **Vite** - Fast development with Vite
- ✅ **All Hooks** - Demonstrates all nuqs-presets hooks

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
