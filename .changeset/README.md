# Changesets

This folder contains changeset files for version management and changelog generation.

## How to use

1. Make your changes to the codebase
2. Run `npm run changeset` to create a changeset
3. Select the type of change (patch, minor, or major)
4. Describe your changes
5. Commit the changeset file along with your changes
6. When ready to release, run `npm run version` to update versions
7. Run `npm run release` to publish to npm

For more information, visit https://github.com/changesets/changesets
